# Extras — Web server support files

These files are deployed to the web root by `install.sh`. Do not modify them directly — run `setup.php` to configure domain and folder settings.

- `index.php` / `globalVariables.php` / `viewer.php` — file browser and markdown viewer placed at the web root
- `PostToAPITemplate.ps1` — template for scripts that POST to an API
- `telemmentryTemplate.php` — telemetry logging template
